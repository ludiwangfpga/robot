# 配置A的End-Effector Position轨迹分析

## 坐标系统定义

```
                  ↓ 期望位置 (xd = -3mm)

     x = +3mm    x = 0mm         x = -3mm
        ●─────────|────────────────●
      起点       墙面              期望深度
   (自由空间)   (接触面)        (目标穿透)
```

### 横坐标 (X轴)：**时间 (Time, ms)**
- 范围：0 ~ 50 毫秒
- 代表整个50ms的运动过程

### 纵坐标 (Y轴)：**位置 (Position, mm)**
- **正值** (x > 0)：离墙面有距离，在自由空间
- **零值** (x = 0)：刚好接触墙面
- **负值** (x < 0)：穿入墙面（接触压缩）

---

## 配置A的运动轨迹

```
时间阶段    位置变化               动作描述
────────────────────────────────────────────

0 ~ 12ms    +3.0mm → +1.28mm     【自由运动阶段】
            ↓                     机械臂以-0.15 m/s接近墙面
            匀速接近              控制器开始输出期望力

12 ~ 20ms   +1.28mm → +0.28mm    【接触初期】
            ↓                     末端轻微接触墙面
            缓慢接近              F_env从0逐步增加

20 ~ 50ms   +0.28mm → -1.45mm    【深度接触阶段】
            ↓                     末端持续穿透
            逐步穿透              穿透深度最大 1.45mm
                                 F_env达到1.8N（峰值）
```

## 物理意义

### 📍 轨迹分段

**第1段 (0~12ms)：**
- 位置：从 +3mm 到 +1.28mm
- 特点：完全在自由空间，无接触
- 机械臂在控制器驱动下逐步接近
- 运动速度约 -0.15 m/s（向下）

**第2段 (12~25ms)：**
- 位置：从 +1.28mm 到 -0.27mm
- 特点：首次接触到穿过墙面
- 接触力从0增加到1.6N
- 穿透速度递减（控制器阻力增大）

**第3段 (25~50ms)：**
- 位置：从 -0.27mm 到 -1.45mm
- 特点：持续穿透，逐步稳定
- 接触力基本稳定在1.8N
- 位置变化缓慢（几近平衡）

## 总结

配置A的End-Effector Position轨迹显示：
- ✅ **平稳的接近**：在自由空间阶段速度恒定
- ✅ **温和的接触**：穿透过程缓和，无剧烈波动
- ✅ **最小穿透**：最终穿透深度仅1.45mm（最短）
- ✅ **低接触力**：1.8N（6个配置中最低）

这解释了为什么**固定Ks=300是这个碰撞场景中最优的选择**。
