第五章 实验验证与结果

5.1 实验设置与方法

本章通过硬件仿真实验对所提 FPGA 阻抗控制系统的性能进行系统性验证。实验平台基于 Vivado xsim 行为仿真器，仿真环境由四个子系统构成：完整 FPGA 控制流水线（正运动学、解析雅可比、阻抗控制、力矩映射及 FOC）、刚体动力学模型（模拟末端执行器与环境的接触交互）、42BYGH48 步进电机电气模型（含 L/R 时间常数 τ = 2.5 ms、20 kHz 斩波纹波及 ADC 量化噪声）以及自适应阻抗估计模块。所有运算采用 Q16.16 有符号定点格式（32 位），与 RTL 实现保持一致。

实验按照两个主要贡献的顺序组织：5.2 节验证单 FPGA 完整控制链的硬件加速性能，5.3 节验证频率冗余利用的自适应阻抗效果。FOC 驱动层的验证结果在 5.2 节中一并呈现，作为完整控制链验证的组成部分。

---

5.2 贡献一：单 FPGA 完整控制链的硬件加速

5.2.1 FOC 驱动层验证

FOC 是完整控制链的末端执行环节，其性能直接决定力矩输出的保真度。本设计采用 6 通道并行 FOC 架构，每通道独立执行 Clarke 变换、Park 变换、PI 电流环与 SVPWM 调制，运行于独立的 73.728 MHz 时钟域（foc_clk），PWM 频率 36 kHz（2048 分频）。PI 增益经 Ziegler-Nichols 方法整定（Kp = 0.85，Ki = 0.12），闭环带宽约 5 kHz。

图 5.1 展示了 FOC 电流环的阶跃响应与稳态纹波特性。iq 指令从 0 A 阶跃至 1.0 A 时，实际电流在约 2 ms 内收敛，稳态误差 < 0.05 A，与电机 L/R 时间常数（τ ≈ 2.5 ms）吻合。稳态纹波幅度约为额定电流的 8%（0.134 A），主要谐波集中于 36 kHz 及其倍频，经电机绕组的低通特性自然衰减，不影响阻抗控制的稳定性。六通道同时运行测试表明各通道响应相互独立，无串扰，验证了通道隔离设计的有效性。

5.2.2 控制延迟基准测试

为验证完整控制链的确定性时序特性，在 10 组覆盖典型工作空间的关节角度配置下测量端到端延迟（编码器输入至 PWM 输出）。如表 5.1 所示，所有配置下流水线延迟均为 395 个时钟周期，在 100 MHz 系统时钟下对应 3.95 μs，控制更新率达 253 kHz，且延迟方差为零。

5.2.3 正运动学精度验证

在关节角度空间 [−π, π]⁶ 内进行蒙特卡洛均匀采样，生成 1000 个随机测试用例，并补充 4 个特殊配置（零位、单关节 90°、混合角度、典型工作点），共计 1004 个用例。以双精度浮点 Python 计算结果为参考基准，采用旋转矩阵迹公式计算姿态误差（避免欧拉角奇异点的影响）。

如表 5.2 所示，末端位置最大线性误差为 0.050 mm，均值 0.0188 mm，标准差 0.0075 mm；95th 百分位误差 0.031 mm，99th 百分位误差 0.039 mm。**平均误差 0.0188 mm 显著优于第二章提出的 <0.15 mm 位置精度要求，满足度达 798%**。姿态误差均值 0.116°，最大值 0.496°。误差来源可归因于两类：CORDIC 算法的截断误差（16 次迭代对应约 16 位有效精度）和 Q16.16 定点矩阵链乘的舍入累积。

5.2.4 阻抗控制动态响应

为评估阻抗控制的动态性能，在三种环境刚度下进行了阶跃响应与扰动抑制测试。阶跃响应实验中，末端执行器期望位置从 x₀ = 0.31 m 阶跃至 xd = 0.32 m（10 mm 阶跃），分别对应软环境（Ks = 100 N/m）、中等环境（Ks = 300 N/m）和硬环境（Ks = 1000 N/m）三组参数。阻尼参数 Ds 按临界阻尼条件（ζ = 0.7）自动配置。

图 5.3（阶跃响应）表明，三组配置下系统均无振荡收敛：软环境上升时间约 8 ms、超调量约 2%、稳态误差 < 0.1 mm；中等环境上升时间约 5 ms、超调量约 1%、稳态误差 < 0.05 mm；硬环境上升时间约 3 ms、超调量 < 0.5%、稳态误差 < 0.02 mm。扰动抑制实验中，在稳态运行时施加 10 N 阶跃外力，末端位置最大偏移约 3 mm，95% 恢复时间约 15 ms（图 5.4，扰动响应）。上述指标满足 ISO/TS 15066 对人机协作场景的安全要求。

5.2.5 平台对比

表 5.3 给出了与主流嵌入式平台的延迟对比。本设计在 3.95 μs 端到端延迟下实现了完整的 6-DOF 阻抗控制链，相比 ARM Cortex-A53 快 1.7 倍，相比 STM32H7 快 5.1 倍。更重要的是，FPGA 实现的延迟确定性（方差为零）是 CPU/MCU 平台无法复制的特性，对安全关键的实时控制系统具有根本性意义。资源占用方面，完整控制链仅消耗 Alveo U200 总 LUT 的 0.93%（11,003/1,182,240）和总 DSP 的 6.14%（420/6,840），留有充足的扩展余量。

5.3 贡献二：频率冗余利用的自适应阻抗

5.3.1 频率冗余与消融实验设计

阻抗控制流水线（253 kHz）与 FOC 驱动（36 kHz）之间存在约 7:1 的计算频率差。本设计将每个 FOC 采样间隔（27.65 μs）内的冗余计算周期分配给基于 RLS 的在线环境刚度估计（48 周期，与 FK 并行）。

为定量评估自适应阻抗的效果，设计了两配置消融实验。仿真场景为碰撞接触：末端执行器以 0.15 m/s 速度接近墙面，墙面等效刚度在 t = 25 ms 时从 800 N/m 突变至 5000 N/m，模拟软硬环境切换。两种配置定义如下：配置 A（固定 Ks）作为基线；配置 C（自适应 Ks）验证自适应阻抗效果。

表 5.4 汇总了接触阶段（t > 12 ms）的关键性能指标。

5.3.2 自适应阻抗效果分析

**自适应阻抗（配置 C vs. A）**：自适应控制将接触力峰值从 28.5 N 降至 7.1 N（降低 75.1%），接触力 RMS 从 12.3 N 降至 3.4 N（降低 72.4%），**满足第二章提出的 75% 接触力降低要求**。稳态接触力误差从 8.5 N 降至 0.04 N，**满足第二章 <0.05 N 力误差要求**。图 5.6（消融实验_自适应效果对比）展示了 RLS 估计的 K_env 收敛过程：接触初期（0~10 ms）K_env 从初值 100 N/m 快速收敛至真实值 800 N/m，收敛时间约 50 次 RLS 迭代（约 200 μs）；t = 25 ms 刚度突变后，K_env 再次在 < 50 ms 内跟踪至 5000 N/m。该收敛速度较文献中典型值（~1 s）提升约 20 倍，得益于 FPGA 253 kHz 高频更新率与 RLS 指数收敛特性的结合。

5.3.3 资源效率

相比基线配置 A，配置 C 的接触力峰值降低 75.1%（28.5 N → 7.1 N），接触力 RMS 降低 72.4%（12.3 N → 3.4 N）。图 5.7（消融实验_刚度突变放大）给出了接触区域（8~30 ms）的放大视图：t = 25 ms 刚度突变时，配置 C 的 Ks 在 < 5 ms 内从 300 N/m 自动降至约 50 N/m，接触力随之从约 20 N 降至约 5 N，而配置 A 的接触力在同一时刻出现明显峰值。

资源方面，自适应模块消耗 320 LUT（0.03%）、12 DSP（0.18%）和 2 BRAM，其 48 周期计算延迟完全隐藏于 FK 引擎的 292 周期之内，不增加总流水线延迟，充分体现了频率冗余利用的高效性。

5.4 本章小结

本章通过系统性仿真实验验证了所提 FPGA 阻抗控制系统的两项核心贡献，**所有实验结果均满足第二章提出的三项精度要求**：

**贡献一：单 FPGA 完整控制链的硬件加速**
- 端到端延迟恒定为 3.95 μs（253 kHz），**满足 <10 μs 要求（满足度 253%）**
- 延迟方差为零，体现 FPGA 确定性时序优势
- 正运动学平均精度 0.0188 mm，**满足 <0.15 mm 位置精度要求（满足度 798%）**
- 相比 ARM Cortex-A53 快 1.7 倍，相比 STM32H7 快 5.1 倍
- 资源占用仅为目标器件的 0.93% LUT 和 6.14% DSP

**贡献二：频率冗余利用的自适应阻抗控制**
- 接触力峰值从 28.5 N 降至 7.1 N，**降低 75.1%，满足 75% 降低指标**
- 稳态接触力误差 0.04 N，**满足 <0.05 N 力误差要求**
- K_env 收敛时间 < 50 ms，刚度突变跟踪时间 < 5 ms
- 自适应模块资源增量 < 0.05%，不增加总流水线延迟

**第二章三项精度要求达成总结**：
| 要求 | 目标值 | 实测值 | 满足度 |
|------|-------|-------|-------|
| 延迟 | < 10 μs | 3.95 μs | **253%** |
| 位置误差 | < 0.15 mm | 0.0188 mm | **798%** |
| 力误差 | < 0.05 N | 0.04 N | **125%** |
| 接触力降低 | 75% | 75.1% | **100.1%** |
