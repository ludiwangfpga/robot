第一章 绪论

1.1 研究背景与意义

双边遥操作系统（Bilateral Teleoperation System）是实现远程精细操作的关键技术，广泛应用于微创手术、核设施维护、深海作业和空间探索等场景[1]。该系统通过主从手机械臂架构，使操作者能够在本地控制远端从手执行任务，同时接收从手与环境交互的力反馈，获得触觉临场感（Telepresence）[2]。

在医疗机器人领域，手术机器人需要满足严苛的精度与安全要求。以机器人辅助微创手术（Robot-Assisted Minimally Invasive Surgery, RMIS）为例，末端执行器的定位精度需达到 0.15 mm 以内[3]，工具与组织的交互力通常处于亚牛顿量级（0.1-0.5 N）[4]。光波导触觉传感器已实现 0.026 N 的力分辨率[5]，而猪肾触诊实验表明正常组织与硬结节的接触力差异仅 0.1 N 左右[6]——这意味着遥操作系统须具备区分正常组织与病变组织的能力，力控制误差一旦超过该阈值，可能对健康组织造成不可逆损伤。

在延迟方面，IEEE 1918.1 触觉互联网标准规定机器对机器远程操作的端到端往返延迟目标为 1 ms[7]。当往返延迟超过该阈值时，力反馈信号与操作者动作之间的相位偏移将导致闭环系统趋于振荡乃至失稳，操作者将感知到明显的"黏滞感"，丧失对远端环境的精确触觉判断[7]。该标准将 1 ms 预算分配给计算、网络传输和机械响应三个环节，分配给单端计算的预算不超过 100 μs。

然而，现有遥操作系统在延迟和精度方面均存在显著局限。传统的机械臂控制系统采用 FPGA + DSP 或 FPGA + MCU 的异构架构[8]，控制链分散在多个处理器上，导致通信延迟和同步复杂性。基于通用处理器的顺序执行模式是触觉远程操作系统的主要延迟来源——已有研究表明，计算密集型算法在定制硬件上可获得相比软件最高三个数量级的加速[9]。此外，固定参数的阻抗控制器在面对软硬环境突变时存在根本矛盾：高刚度参数在自由空间运动时可保证良好的位置跟踪精度，但在硬接触时会产生过大的接触力；反之，低刚度参数虽可限制接触力，却导致软环境中的位置跟踪误差显著增大[10]。

因此，研究具备低延迟、高精度、自适应能力的单芯片遥操作控制方案，对于提升远程手术的安全性和精准度具有重要意义。

1.2 国内外研究现状

1.2.1 FPGA 在机器人控制中的应用

现场可编程门阵列（Field-Programmable Gate Array, FPGA）因其硬件并行性和确定性时序特性，在机器人实时控制领域得到广泛应用。Monmasson 等[11]综述了 FPGA 在工业控制中的应用，指出其时钟级确定性延迟特性对于安全关键系统至关重要。

在机械臂控制方面，Huang 等[12]提出了 DSP/FPGA 异构架构的自适应笛卡尔阻抗控制方案，实现了 4-DOF 空间机械臂的控制，延迟约 200 μs。Phan 等[13]基于 FPGA 实现了 6-DOF 关节机器人的扭矩控制，但未集成完整的控制链（运动学、力控制、电机驱动），且不具备自适应能力。现有方案普遍采用多芯片异构架构，导致通信延迟和同步复杂性，无法满足 <10 μs 的严苛延迟要求。

1.2.2 自适应阻抗控制研究进展

自适应阻抗控制通过在线估计环境刚度并动态调整控制器参数，解决了固定参数在软硬环境切换时的矛盾。Abu-Dakka 与 Kyrki[1]综述了变阻抗控制与学习的研究进展，指出环境适应性是人机交互安全性的关键。Roveda 等[10]提出了基于递归最小二乘（Recursive Least Squares, RLS）的环境刚度估计方法，并采用反向自适应规则（硬接触时降低控制器刚度）将接触力限制在安全范围内。

然而，现有自适应方案多基于 CPU/DSP 实现，更新频率通常为 1 kHz 量级[14]，难以满足高频力控制的需求。此外，自适应算法的计算开销（矩阵求逆、除法等）在传统处理器上形成瓶颈，限制了其实时性能。

1.2.3 现有研究的不足

综上所述，现有研究存在以下不足：

（1）延迟瓶颈：异构架构的通信延迟使现有方案难以满足 <100 μs 的单端计算预算要求；

（2）控制链分散：运动学、力控制、电机驱动分布在不同芯片，缺乏单片集成的完整解决方案；

（3）自适应能力不足：现有 FPGA 方案多为固定参数，自适应方案更新频率低（~1 kHz），无法充分利用 FPGA 的高频计算能力；

（4）频率冗余浪费：阻抗控制与电机驱动之间存在显著频率差（通常 5:1 至 10:1），现有方案未充分利用这一冗余资源。

1.3 主要研究内容与贡献

针对上述不足，本文提出了一种基于 FPGA 的 6-DOF 机械臂双边遥操作系统，主要贡献如下：

**贡献一：单 FPGA 完整控制链的硬件加速**

本文在单片 FPGA（Xilinx Alveo U200）上实现了从编码器输入到 PWM 输出的完整控制链，包括正运动学（Forward Kinematics, FK）、解析雅可比、速度估计、阻抗控制、力矩映射和磁场定向控制（Field-Oriented Control, FOC）。通过并行化 FK 和雅可比计算，将端到端延迟降至 3.95 μs（253 kHz 更新率），满足 IEEE 1918.1 标准提出的 <100 μs 计算预算要求，相比 ARM Cortex-A53 快 38 倍，相比 STM32H7 快 101 倍。该设计的关键优势在于确定性时序：无论关节角度如何变化，流水线延迟始终恒定，不受缓存未命中、中断或操作系统调度的影响。

**贡献二：频率冗余利用的反向自适应阻抗控制**

本文发现阻抗控制（253 kHz）与 FOC 驱动（36 kHz）之间存在 7:1 的频率冗余。利用这一冗余，在不增加硬件成本的前提下实现了基于 RLS 的在线环境刚度估计与反向自适应阻抗控制。反向自适应规则使控制器刚度与环境刚度成反比（硬接触时降低刚度），将接触力峰值从 28.5 N 降至 7.1 N（降低 75.1%），满足 <0.05 N 力误差要求。RLS 算法在约 50 次迭代（约 200 μs）内收敛至环境刚度真实值的 95% 以内，较文献典型值（~1 s）提升约 5000 倍。

**贡献三：FPGA + ROS2 分层架构的可扩展设计**

本文采用分层架构：FPGA 负责实时控制（253 kHz），ROS2 负责高层规划（~10 Hz）。通过 AXI-Lite 寄存器接口，上位机可实时配置控制参数（K_s、D_s、期望位姿等），实现了硬件加速与软件灵活性的结合。

本文的核心创新在于**首次在单片 FPGA 上完整实现了反向自适应阻抗控制链**，充分利用频率冗余，在不增加硬件资源的前提下同时提升了力控制精度（接触力降低 75.1%）和环境适应性，为低延迟、高精度、可扩展的机器人力反馈系统提供了新的硬件加速范式。

1.4 论文组织结构

本文共分为六章，组织结构如下：

第一章（绪论）：介绍研究背景、国内外研究现状、主要贡献。

第二章（双边遥操作系统架构）：阐述系统总体架构、精度要求（延迟 <10 μs、位置误差 <0.15 mm、力误差 <0.05 N 且接触力降低 75%）、控制链路数据流。

第三章（控制算法原理）：介绍正运动学、解析雅可比、速度估计、阻抗控制、力矩映射的算法原理与数学模型。

第四章（FPGA 硬件加速实现）：详细阐述各算法的硬件实现方案，包括反向自适应阻抗控制的 RTL 设计。

第五章（实验验证与结果）：通过系统性仿真实验验证系统的延迟、精度、自适应性能。

第六章（结论与展望）：总结主要贡献，讨论局限性与未来研究方向。
